# Lietotāja rokasgrāmata

## Kā atvērt projektu
1. Atver interneta pārlūkprogrammā projekta publisko lapu:  
   **https://niks15022.github.io/it-arodbiedriba-prakse/**  
2. Lapa automātiski ielādēs Supabase datus un parādīs tos sarakstā.

## Ko lietotājs redz
- Virsrakstu “Supabase dati”
- Sarakstu ar ierakstiem no tabulas “members”

Šajā projekta versijā dati ir tikai **apskatāmi**, bez iespējas tos labot vai pievienot jaunus.

## Ko darīt, ja kaut kas nestrādā
- Pārlādē lapu (Ctrl + R)
- Pārbaudi vai ir interneta savienojums
- Pārbaudi, vai Supabase projekts nav izslēgts (tas var notikt bezmaksas plānā)
- Atver pārlūkprogrammas konsoli (F12 → Console)
